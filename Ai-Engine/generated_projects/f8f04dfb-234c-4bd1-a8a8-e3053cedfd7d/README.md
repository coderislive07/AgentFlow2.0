# Portfolio Website

This is a portfolio website built with HTML, CSS, JavaScript, and Tailwind CSS.

## Features
- Responsive design
- Contact form
- Project showcase

## Installation
1. Clone the repository
2. Run `npm install`
3. Run `npm start` to start the development server

## Deployment
1. Run `npm run build` to create a production build
2. Deploy the `dist` folder to your hosting provider
